详细说明​

数学公式​

支持复杂数学公式的计算，可以使用变量和各种数学函数。

使用场景：复杂计算、公式运算等。

针脚说明：

  • 公式：要计算的公式表达式，公式内的变量仅支持单个字母。
    • 运算符：
      • +：加法。
      • -：减法。
      • *****：乘法。
      • /：除法。
      • ^：幂运算。
      • %：取余数。
    • 函数：⨍(x)
      • abs：取绝对值。
      • acos：求反余弦。
      • asin：求反正弦。
      • atan：求反正切。
      • cbrt：求立方根。
      • ceil：向上取整。
      • cos：求余弦。
      • cosh：求双曲余弦。
      • exp：求指数。
      • floor：向下取整。
      • log：求对数。
      • log2：求对数。
      • log10：求对数。
      • sin：求正弦。
      • sinh：求双曲正弦。
      • sqrt：求平方根。
      • tan：求正切。
      • tanh：求双曲正切。
      • signum：求符号。
  • 结果：输出的计算结果。

数值相加​

将多个数值相加求和。

使用场景：累加统计、总和计算等 。

针脚说明：

  • 数值(1)：要相加的多个数值之一。
  • 数值(2)：要相加的多个数值之一。
  • 添加针脚：添加新的数值。
  • 数值(右)：输出的和。

  • 数值(右)：输出的差值。

取整​

将数值按指定规则取整（向上取整、向下取整、四舍五入等）。

使用场景：数值处理、精度控制等。

针脚说明：

  • 数值：要取整的数值。
  • 取整方式：指定取整方式（向上取整、向下取整、四舍五入、取整数部分等）。
    • 四舍五入：将数值四舍五入。
    • 向上取整：将数值向上取整，向数轴正方向取整。
    • 向下取整：将数值向下取整，向数轴负方向取整。
  • 整数：输出的取整后数值。
