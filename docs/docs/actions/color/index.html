颜色

颜色处理类动用于获取、识别和操作屏幕上的颜色。

动作列表​

动作类型功能说明
获取颜色获取图片指定位置颜色
识别颜色识别指定颜色在图片中的所有区域
颜色是否存在判断指定颜色是否存在
颜色比较判断两个颜色是否相似
点击颜色点击指定颜色

详细说明​

获取颜色​

获取图片中指定位置的像素颜色。

使用场景：读取界面元素颜色、颜色检测等。

针脚说明：

  • 图片：要获取颜色的图片。
  • 位置：指定要获取颜色的像素位置。
  • 颜色：输出获取的颜色值。

识别颜色​

在图片中查找所有与指定颜色相似的区域。

使用场景：批量查找特定颜色位置、界面元素定位等。

针脚说明：

  • 超时：等待超时时间（毫秒）。
  • 检查间隔：识别颜色间隔时间（毫秒）。
  • 图片：要搜索的图片，默认为当前屏幕。
  • 颜色：要查找的颜色值。
  • 相似度：颜色匹配的相似度。
  • 区域列表：输出所有匹配的区域，按面积降序排列。
  • 首个区域：输出第一个匹配的区域，也是颜色面积最大的区域。

颜色是否存在​

判断图片中是否存在指定颜色。

使用场景：条件判断、界面状态检测等。

针脚说明：

  • 图片：要搜索的图片，默认为当前屏幕。
  • 颜色：要查找的颜色值。
  • 相似度：颜色匹配的相似度。
  • 结果：输出是否找到指定颜色。

颜色比较​

比较两个颜色是否相似。

使用场景：颜色判断、条件分支等。

针脚说明：

  • 颜色(1)：要对比的颜色之一。
  • 颜色(2)：要对比的颜色之一。
  • RGB偏移：RGB颜色偏移量，在偏移值内则认为颜色相似。
  • 结果：输出颜色是否相似。

点击颜色​

在屏幕上找到指定颜色并点击。

使用场景：点击特定颜色的按钮或图标。

针脚说明：

  • 颜色：要查找的颜色值。
  • 相似度：颜色匹配的相似度。
  • 区域限制：指定搜索的屏幕区域。
  • 随机偏移：是否启用随机偏移，启用后会点击颜色区域的随机位置。
  • 点击所有：是否点击所有匹配的区域。
  • 执行(底部)：找到颜色后并正确点击之后执行的分支。
  • 未达成：未找到颜色时执行的分支。
