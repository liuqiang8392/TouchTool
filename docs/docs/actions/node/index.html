详细说明​

查找控件​

根据指定条件（文本、ID、类名等）查找界面上的控件。

使用场景：定位按钮、文本框等UI元素。

身份方式文本方式ID方式正则路径类名方式描述方式

针脚说明：

  • 超时：等待超时时间（毫秒）。
  • 检查间隔：查找控件间隔时间（毫秒）。
  • 方式：选择查找方式。
    • 身份：根据控件身份查找控件。
      • 身份：控件身份。
      • 精确身份：是否启用精确匹配，模糊身份会尽可能给出一个匹配结果。
      • 控件：输出找到的控件对象。
    • 文本：根据文本查找控件。
      • 文本：控件文本。
      • 区域：查找控件区域。
      • 控件：输出找到的第一个控件对象。
      • 控件列表：输出找到的所有控件的列表。
    • ID：根据ID查找控件。
      • ID：控件ID。
      • 区域：查找控件区域。
      • 控件：输出找到的第一个控件对象。
      • 控件列表：输出找到的所有控件的列表。
    • 正则路径：根据控件路径正则表达式查找控件。
      • 正则路径：控件路径正则表达式。
      • 控件：输出找到的第一个控件对象。
      • 控件列表：输出找到的所有控件的列表。
    • 类名：根据类名查找控件。
      • 类名：控件类名。
      • 区域：查找控件区域。
      • 控件：输出找到的第一个控件对象。
      • 控件列表：输出找到的所有控件的列表。
    • 描述：根据描述查找控件。
      • 描述：控件描述。
      • 区域：查找控件区域。
      • 控件：输出找到的第一个控件对象。
      • 控件列表：输出找到的所有控件的列表。

控件是否存在​

判断符合指定条件的控件是否存在。

使用场景：条件判断，检查界面元素状态。

身份方式文本方式ID方式正则路径类名方式描述方式

针脚说明：

  • 方式：选择查找方式。
    • 身份：根据控件身份查找控件。
      • 身份：控件身份。
      • 精确身份：是否启用精确匹配，模糊身份会尽可能给出一个匹配结果。
    • 文本：根据文本查找控件。
      • 文本：控件文本。
      • 区域：查找控件区域。
    • ID：根据ID查找控件。
      • ID：控件ID。
      • 区域：查找控件区域。
    • 正则路径：根据控件路径正则表达式查找控件。
      • 正则路径：控件路径正则表达式。
    • 类名：根据类名查找控件。
      • 类名：控件类名。
      • 区域：查找控件区域。
    • 描述：根据描述查找控件。
      • 描述：控件描述。
      • 区域：查找控件区域。
  • 结果：输出控件是否存在。

获取区域内控件​

获取与指定区域相交的所有控件。

使用场景：批量获取某个区域内的控件。

针脚说明：

  • 区域：指定的屏幕区域。
  • 控件列表：输出区域内所有控件的列表。
  • 最上层控件：输出最上层的控件对象。

获取控件信息​

获取控件的详细信息，包括文本、位置、大小等。

使用场景：读取控件属性，获取界面数据。

针脚说明：

  • 控件：要获取信息的控件对象。
  • 类名：输出控件的类名。
  • 控件ID：输出控件的ID。
  • 文本：输出控件的文本。
  • 描述：输出控件的描述。
  • 区域：输出控件的区域。
  • 是否可用：输出控件是否可点击。
  • 是否可见：输出控件是否可见。
  • 控件身份：输出控件的控件身份。

获取子控件​

获取指定控件的所有子控件。

使用场景：遍历控件树，获取嵌套控件。

针脚说明：

  • 控件：要获取子控件的父控件对象。
  • 包括子孙：是否包括所有子孙控件。
  • 控件列表：输出所有子控件的列表。

获取父控件​

获取指定控件的父控件。

使用场景：向上查找控件层级。

针脚说明：

  • 控件(左)：要获取父控件的子控件对象。
  • 控件(右)：输出父控件对象。

获取所有窗口​

获取当前界面上的所有窗口。

使用场景：多窗口场景下的窗口管理。

针脚说明：

  • 控件列表：输出所有窗口对象的列表。

是有效控件?​

检查控件是否仍然有效（未被销毁）。

使用场景：操作前验证控件有效性。

针脚说明：

  • 控件：要检查的控件对象。
  • 结果：输出控件是否有效。

点击控件​

对指定控件执行点击操作。

使用场景：点击按钮、链接等。

针脚说明：

  • 控件：要点击的控件对象。
  • 长按：是否长按控件。
  • 精确点击：无论要点击的控件是否可以点击，都强制进行点击，否则查找可点击的父控件进行点击。
  • 执行(底部)：控件正确点击时执行的分支。
  • 未达成：控件未能点击时执行的分支。

输入框输入文本​

向文本输入框中输入指定文本。

使用场景：填写表单、搜索框输入等。

针脚说明：

  • 输入框：目标输入框控件对象，默认会使用当前界面第一个输入框控件。
  • 文本：要输入的文本内容。
  • 追加：是否追加输入。
  • 尝试模拟回车：是否尝试模拟回车键。
  • 执行(底部)：输入成功时执行的分支。
  • 未达成：输入失败时执行的分支。

输入框粘贴​

将剪贴板内容粘贴到输入框中。

使用场景：粘贴复制的文本到输入框。

针脚说明：

  • 输入框：目标输入框控件对象，默认会使用当前界面第一个输入框控件。
  • 尝试模拟回车：是否尝试模拟回车键。
  • 执行(底部)：输入成功时执行的分支。
  • 未达成：输入失败时执行的分支。

手动选择控件​

通过用户交互方式选择一个控件。

使用场景：需要用户指定操作目标。

针脚说明：

  • 控件：输出用户选中的控件对象。
  • 执行(底部)：用户选择控件时执行的分支。
  • 未达成：用户取消选择时执行的分支。
