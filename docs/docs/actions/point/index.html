位置

位置操作类动用于创建和操作屏幕上的坐标点。

动作列表​

动作类型功能说明
位置转数值提取位置的横纵坐标
数值转位置将数值转换为位置
位置偏移将位置进行偏移
位置转为手势将位置转换为手势
位置列表转为手势将位置列表转换为手势
执行手势执行指定手势
点击位置点击指定位置

详细说明​

位置转数值​

将位置对象转换为X和Y坐标数值。

使用场景：提取坐标进行计算或判断。

针脚说明：

  • 位置：要转换的位置对象。
  • X：输出位置的X坐标值。
  • Y：输出位置的Y坐标值。

数值转位置​

使用X和Y坐标创建位置对象。

使用场景：根据坐标动态创建位置。

针脚说明：

  • X：位置的X坐标值。
  • Y：位置的Y坐标值。
  • 位置：输出创建的位置对象。

位置偏移​

将位置在水平和垂直方向上移动指定距离。

使用场景：调整位置、相对偏移等。

针脚说明：

  • 位置：要偏移的原位置。
  • 偏移：位置的偏移距离。
  • 结果：输出偏移后的位置。

位置转为手势​

将单个位置转换为点击手势。

使用场景：创建点击或长按手势。

针脚说明：

  • 位置：手势的开始位置。
  • 触摸时间：手势中到下个位置的时间（毫秒）。
  • 添加针脚：添加一对位置和持续时间的输入针脚。
  • 手势：输出创建的手势对象。

位置列表转为手势​

将多个位置转换为滑动手势。

使用场景：创建滑动、拖拽等连续手势。

针脚说明：

  • 位置列表：手势经过的位置列表。
  • 触摸总时间：手势执行的总时长（毫秒）。
  • 手势：输出创建的手势对象。

执行手势​

执行指定的手势操作。

使用场景：执行点击、滑动、长按等手势。

针脚说明：

  • 手势：要执行的手势对象。
  • 手势速度：手势执行速度，按倍率缩放手势总时间。
  • 随机偏移：手势 上下左右随机偏移最大距离。

点击位置​

在指定坐标位置执行点击操作。

使用场景：点击屏幕特定位置。

针脚说明：

  • 位置：点击的位置。
  • 触摸时间：点击的持续时间（毫秒）。
  • 随机偏移：点击位置上下左右随机偏移最大距离。
  • 点击方式：
    • 无障碍：使用无障碍点击，会被软件检测和拦截。
    • Shell：使用Shell点击，模拟触摸屏点击，但是需要超级用户权限。
