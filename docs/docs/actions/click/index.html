点击

点击类动作用于执行各种点击和手势操作。

动作列表​

动作类型功能说明
执行手势执行指定手势
点击位置点击指定位置
点击控件点击传入的控件
点击图片点击目标图片
点击颜色点击指定颜色

详细说明​

执行手势​

执行指定的手势操作。

使用场景：执行点击、滑动、长按等手势。

针脚说明：

  • 手势：要执行的手势对象。
  • 手势速度：手势执行速度，按倍率缩放手势总时间。
  • 随机偏移：手势上下左右随机偏移最大距离。

点击位置​

在指定坐标位置执行点击操作。

使用场景：点击屏幕特定位置。

针脚说明：

  • 位置：点击的位置。
  • 触摸时间：点击的持续时间（毫秒）。
  • 随机偏移：点击位置上下左右随机偏移最大距离。
  • 点击方式：
    • 无障碍：使用无障碍点击，会被软件检测和拦截。
    • Shell：使用Shell点击，模拟触摸屏点击，但是需要超级用户权限。

点击控件​

对指定控件执行点击操作。

使用场景：点击按钮、链接等。

针脚说明：

  • 控件：要点击的控件对象。
  • 长按：是否长按控件。
  • 精确点击：无论要点击的控件是否可以点击，都强制进行点击，否则查找可点击的父控件进行点击。
  • 执行(底部)：控件正确点击时执行的分支。
  • 未达成：控件未能点击时执行的分支。

点击图片​

在原图中找到目标图片并点击。

使用场景 ：点击找到的按钮、图标等。

针脚说明：

  • 目标：要查找的目标图片。
  • 相似度：指定匹配的相似度阈值。
  • 区域限制：指定图片查找区域。
  • 缩放加速：是否启用缩放加速，启用后图片会缩小匹配，以提升识别效率，但是会导致相似度降低。
    • 原图：原图。
    • 两倍：两倍缩放，变为原图四分之一大小。
    • 四倍：四倍缩放，变为原图十六分之一大小。
  • 去色加速：是否启用去色加速，启用后图片会去色匹配，以提升识别效率，但是会导致相似度降低。
  • 随机偏移：是否启用随机偏移，启用后会点击图片的随机位置。
  • 执行(底部)：找到图片后并正确点击之后执行的分支。
  • 未达成：未找到图片时执行的分支。

点击颜色​

在屏幕上找到指定颜色并点击。

使用场景：点击特定颜色的按钮或图标。

针脚说明：

  • 颜色：要查找的颜色值。
  • 相似度：颜色匹配的相似度。
  • 区域限制：指定搜索的屏幕区域。
  • 随机偏移：是否启用随机偏移，启用后会点击颜色区域的随机位置。
  • 点击所有：是否点击所有匹配的区域。
  • 执行(底部)：找到颜色后并正确点击之后执行的分支。
  • 未达成：未找到颜色时执行的分支。
