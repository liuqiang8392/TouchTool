条件

条件判断类动作用于组合和处理布尔条件，实现复杂的逻辑判断。

动作列表​

动作类型功能说明
所有条件达成所有条件达成结果才是达成
任意条件达成任意条件达成结果就能达成
反转条件取与条件相反的结果
短路所有条件达成短路所有条件达成结果才是达成（遇到false则停止）
短路任意条件达成短路任意条件达成结果就能达成（遇到true则停止）

详细说明​

所有条件达成​

逻辑AND操作，当所有输入条件都为真时返回真。

使用场景：需要同时满足多个条件时使用。

针脚说明：

  • 条件(1)：要判断的多个条件之一。
  • 条件(2)：要判断的多个条件之一。
  • 添加针脚：添加新的条件。
  • 结果：所有条件为真时结果为真。

任意条件达成​

逻辑OR操作，当任意一个输入条件为真时返回真。

使用场景：只需满足其中一个条件时使用。

针脚说明：

  • 条件(1)：要判断的多个条件之一。
  • 条件(2)：要判断的多个条件之一。
  • 添加针脚：添加新的条件。
  • 结果：任意条件为真时结果为真。

反转条件​

逻辑NOT操作，将条件结果取反。

使用场景：需要反向判断时使用。

针脚说明：

  • 条件：输入的布尔条件。
  • 结果：输出布尔值，条件为真时返回假，条件为假时返回真。

短路所有条件达成​

与"所有条件达成"类似，但采用短路求值策略。当遇到第一个false条件时立即停止计算 后续条件。

使用场景：多个条件判断时，希望遇到false立即停止以提高效率。

针脚说明：

  • 条件(1)：要判断的多个条件之一。
  • 条件(2)：要判断的多个条件之一。
  • 添加针脚：添加新的条件。
  • 结果：任意条件为假时结果直接为假，否则返回真。

短路任意条件达成​

与"任意条件达成"类似，但采用短路求值策略。当遇到第一个true条件时立即停止计算后续条件。

使用场景：多个条件判断时，希望遇到true立即停止以提高效率。

针脚说明：

  • 条件(1)：要判断的多个条件之一。
  • 条件(2)：要判断的多个条件之一。
  • 添加针脚：添加新的条件。
  • 结果：任意条件为真时结果直接为真，否则返回假。
