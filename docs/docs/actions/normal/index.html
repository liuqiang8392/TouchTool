通用

通用动作提供常用的基础功能，包括延时、日志、贴图等。

动作列表​

动作类型功能说明
延时等待一段时间后再继续执行
输出日志输出日志到日志界面
贴图到屏幕上将针脚内容贴到屏幕上
关闭贴图关闭对应贴图
关闭所有贴图关闭屏幕所有贴图
配置面板手动输入任务运行需要的配置
停止任务停止对应任务
区域备注对一片区域内的卡片进行备注

详细说明​

延时​

暂停任务执行指定的时间，用于控制任务节奏。

使用场景：等待页面加载、控制执行节奏等。

针脚说明：

  • 时长：等待的时间区间（毫秒），随机选取一个值。

输出日志​

将指定内容输出到日志界面，用于调试和记录。

使用场景：调试任务、记录执行过程等。

针脚说明：

  • 日志：要输出的日志文本。
  • 弹出提示：是否弹出提示框。
  • 窗相对屏位置：指定悬浮窗锚点的相对屏幕锚点的位置。
  • 悬浮窗锚点：指定悬浮窗锚点。
  • 屏幕锚点：指定屏幕锚点。
  • 保存日志：是否保存日志。

贴图到屏幕上​

将图片或内容显示在屏幕上的指定位置。

使用场景：显示提示信息、显示操作指引等。

针脚说明：

  • 对象：要显示的图片或文本内容。
  • 窗相对屏位置：指定悬浮窗锚点的相对屏幕锚点的位置。
  • 悬浮窗锚点：指定悬浮窗锚点。
  • 屏幕锚点：指定屏幕锚点。
  • 标志：贴图标志，用于关闭贴图。

关闭贴图​

关闭指定的屏幕贴图。

使用场景：清理屏幕显示。

针脚说明：

  • 标志：要关闭的贴图标志。

关闭所有贴图​

一次性关闭屏幕上所有贴图。

使用场景：任务结束时清理屏幕。

配置面板​

创建一个配置面板，让用户在任务运行前输入需要的参数。

使用场景：需要用户输入参数的任务。

记忆位置手动选择面板界面

针脚说明：

  • 位置：选择窗口显示位置。
    • 记忆位置：显示在记忆位置的窗口，用户选择后记住位置。
    • 手动选择：设置自定义位置。
      • 窗相对屏位置：指定悬浮窗锚点的相对屏幕锚点的位置。
      • 悬浮窗锚点：指定悬浮窗锚点。
      • 屏幕锚点：指定屏幕锚点。
  • 保存运行参数：是否保存运行参数，下次运行时自动读取。
  • 自动确认等待：超过等待时间（毫秒）后自动确认当前配置。

停止任务​

停止指定的任务运行。

使用场景：条件满足时终止任务。

针脚说明：

  • 除自身所有任务：
    • 是：停止除自身以外的所有任务。
    • 否：
      • 任务：要停止的任务。

区域备注​

对蓝图上某个区域内的卡片添加文字备注。

使用场景：记录任务逻辑、添加说明等。
