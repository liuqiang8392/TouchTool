开始

开始动作用于定义任务的启动条件，当满足特定条件时自动执行任务。

动作列表​

动作类型功能说明
手动执行在选中应用内显示执行悬浮窗
进应用或界面时执行进入对应应用或界面时自动执行
退出应用时执行退出对应应用时自动执行
定时执行在设定的时间执行
收到新通知时执行收到新通知或Toast时执行
网络状态变更时执行WIFI、数据、VPN变更时执行
电池状态变更时执行电量变更或充电变更时执行
屏幕状态变更时执行屏幕亮起关闭解锁时执行
蓝牙连接变更时执行蓝牙设备连接或断开时执行
收到分享时执行接受分享行为，分享的内容将自动转换为对应参数
收到广播时执行收到指定广播时自动执行
外部调用外部调用任务，能自动将传入的参数转换到任务内同名变量中去

详细说明​

通用针脚​

针脚说明：

  • 启用：启用或禁用此动作。
  • 重入策略：
    • 重新执行：先关闭当前动作开始的任务，再开始执行。
    • 放弃执行：当前动作开始的任务正在运行时，将放弃本次执行。
    • 并行执行：同时开始多个任务，多个任务之间互不影响。
  • 中断条件：任务运行时会频繁判断中断条件，满足条件时将停止任务。
  • 执行时显示：控制任务执行时是否在悬浮窗显示执行状态。

手动执行​

在选中的应用内显示一个按钮悬浮窗，用户点击对应按钮后执行任务。

使用场景：需要用户主动触发的自动化任务。

集合显示单独显示

针脚说明：

  • 应用或界面：指定需要触发任务的应用或界面。
  • 显示模式：
    • 集合显示：显示一个按钮悬浮窗，所有集合显示按钮都在这个悬浮窗内显示。
      • 自动展开：当显示悬浮窗时，自动展开悬浮窗，显示所有按钮。
    • 单独显示：在对应位置显示一个按钮。
      • 窗相对屏位置：指定悬浮窗锚点的相对屏幕锚点的位置。
      • 悬浮窗锚点：指定悬浮窗锚点。
      • 屏幕锚点：指定屏幕锚点。
      • 锁定位置：锁定按钮位置，按钮不再能移动。

进应用或界面时执行​

当用户打开指定的应用或进入特定界面时，自动触发任务执行。

使用场景：自动签到、自动填写表单等。

针脚说明：

  • 应用或界面：指定需要触发任务应用的包名或界面。
  • 自动中断：当退出对应应用或界面时，自动中断任务。

退出应用时执行​

当用户退出指定应用时，自动触发任务执行。

使用场景：清理数据、保存状态等。

针脚说明：

  • 应用或界面：指定需要触发任务应用的包名或界面。

定时执行​

在设定的时间点自动执行任务。

使用场景：定时提醒、定时备份等。

针脚说明：

  • 日期：指定任务执行日期。
  • 时间：指定任务执行时间。
  • 间隔：指定重复执行间隔时间。

收到新通知时执行​

当系统收到新的通知或Toast消息时触发任务。

使用场景：自动处理特定通知、消息转发等。

针脚说明：

  • 应用或界面：指定需要触发任务的应用。
  • 通知类型：
    • 通知：普通通知。
    • Toast：Toast消息。
  • 通知应用：发送通知的应用。
  • 通知文本：通知的内容。
  • 通知详情：通知内具体信息，包含一些不在通知栏显示的信息。

网络状态变更时执行​

当WIFI、移动数据或VPN连接状态发生变化时触发任务。

使用场景：网络切换时的自动化操作。

针脚说明：

  • 网络：网络类型，可选值如下。
    • 无网络：无网络连接。
    • 移动数据：移动数据连接。
    • WIFI：WIFI连接。
    • VPN：VPN连接。

电池状态变更时执行​

当电量变化或充电状态改变时触发任务。

使用场景：低电量提醒、充电完成通知等。

针脚说明：

  • 状态：电池状态，可选值如下。
    • 未知：未知状态。
    • 充电中：正在充电。
    • 放电中：正在充电，但是充电小于放电的速度。
    • 未充电：未充电。
    • 充满了：充电完成。
  • 电量：当前电量百分比。

屏幕状态变更时执行​

当屏幕亮起、关闭或解锁时触发任务。

使用场景：锁屏时自动清理、解锁时自动执行等。

针脚说明：

  • 状态：屏幕状态，可选值如下。
    • 息屏：息屏状态。
    • 锁屏：亮屏但未解锁。
    • 解锁：已解锁状态。

蓝牙连接变更时执行​

当 蓝牙设备连接或断开时触发任务。

使用场景：连接耳机时自动播放音乐等。

针脚说明：

  • 设备名：蓝牙设备名称。
  • 连接或断开：触发任务时的状态。

收到分享时执行​

当应用接收到来自其他应用的分享内容时触发任务。

使用场景：自动处理分享的文本、图片等。

针脚说明：支持动态针脚，会将内容自动解析到对应类型的针脚中。

收到广播时执行​

当收到指定的系统广播时触发任务。

使用场景：响应系统事件的自动化操作。

针脚说明：

  • 广 播动作：指定需要触发任务的广播。
  • 参数：广播数据。

外部调用​

允许其他应用或任务调用此任务，并自动将传入的参数转换为任务内的同名变量。

使用场景：任务模块化、参数传递等。

针脚说明：

  • 链接：可复制的链接，用于其他应用或任务调用此任务。
    • 参数：支持动态参数，用&拼接，类似“复制的链接&key1=value1&key2=value2”，会根据参数名称解析到对应的针脚中，如果任务中有对应自有变量，也会对这些变量进行填充。
  • 快捷方式：创建桌面快捷方式。
