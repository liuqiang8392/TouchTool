字典

字典操作类动用于创建和操作键值对数据结构。

动作列表​

动作类型功能说明
新建字典新建一个字典
字典长度获取字典长度
字典是否为空判断字典是否为空
字典是否包含键判断字典是否包含键
字典是否包含值判断字典是否包含值
添加或替换值向字典中添加或替换值
移除键值对从字典中移除键值对
字典拼接将两个字典拼接
清空字典清空字典
获取值从字典中获取值
获取所有键获取字典中的所有键
获取所有值获取字典中的所有值
遍历字典遍历字典中的每个键值对

详细说明​

新建字典​

创建一个新的字典。

使用场景：初始化字典数据结构。

针脚说明：

  • 键：要添加的键名。
  • 值：要添加的值。
  • 添加针脚：添加键值对。
  • 字典：输出创建的字典。

字典长度​

获取字典中键值对的数量。

使用场景：判断字典大小、条件判断等。

针脚说明：

  • 对象-对象：要获取长度的字典。
  • 整数：输出字典的键值对数量。

字典是否为空​

判断字典是否不包含任何键值对。

使用场景：空值检查、条件判断等。

针脚说明：

  • 对象-对象：要判断的字典。
  • 结果：字典是否为空。

字典是否包含键​

判断字典中是否包含指定的键。

使用场景：键存在性检查、条件判断等。

针脚说明：

  • 对象-对象：要搜索的字典。
  • 对象：要查找的键名。
  • 结果：字典是否包含该键。

字典是否包含值​

判断字典中是否包含指定的值。

使用场景：值存在性检查、条件判断等。

针脚说明：

  • 对象-对象：要搜索的字典。
  • 对象：要查找的值。
  • 结果：字典是否包含该值。

添加或替换值​

向字典中添加新的键值对，如果键已存在则替换其值。

使用场景：更新字典数据、添加配置等。

针脚说明：

  • 对象-对象：要操作的字典。
  • 键：要添加或修改的键。
  • 值：要设置的值。

移除键值对​

从字典中移除指定键的键值对。

使用场景：删除不需要的数据。

针脚说明：

  • 对象-对象：要操作的字典。
  • 键：要移除的键。
  • 之前的值：输出被移除的键值对中的值。

字典拼接​

将两个字典合并成一个字典。

使用场景：合并多个字典数据。

针脚说明：

  • 对象-对象(1)：第一个字典。
  • 对象-对象(2)：第二个字典。
  • 结果：输出合并后的字典，如果两个字典有相同的键，后面的字典会覆盖前面的值。

清空字典​

移除字典中的所有键值对。

使用场景：重置字典、释放数据等。

针脚说明：

  • 对象-对象：要清空的字典。

获取值​

根据键获取字典中对应的值。

使用场景：读取字典中的数据。

针脚说明：

  • 对象-对象：要获取值的字典。
  • 键：要获取值的键名。
  • 值：输出获取的值。
  • 结果：是否正确获取到值。

获取所有键​

获取字典中所有的键。

使用场景：遍历字典、获取键列表等。

针脚说明：

  • 对象-对象：要获取键的字典。
  • 键：输出字典中所有的键。

获取所有值​

获取字典中所有的值。

使用场景：遍历字典、获取值列表等。

针脚说明：

  • 对象-对象：要获取值的字典。
  • 值：输出字典中所有的值。

遍历字典​

对字典中的每个键值对执行指定操作。

使用场景：批量处理字典数据。

针脚说明：

  • 对象-对象：要遍历的字典。
  • 键：当前遍历的键名。
  • 值：当前遍历的值。
  • 执行(底部)：连接要循环执行的逻辑。
  • 完成：循环完成后执行的分支。
