区域

区域操作类动用于创建、操作和判断屏幕上的矩形区域。

动作列表​

动作类型功能说明
区域转数值提取区域的上下左右数值
数值转区域将数值转换为区域
区域偏移将区域进行偏移
位置是否在区域里判断一个位置是否在指定区域内
区域关系判断两个区域是否包含、相交、不相交
取区域交集取两个区域相交的区域
取区域中心取指定区域的中心位置
取区域随机位置取指定区域的随机位置
手动选择区域选择一个区域
标记区域手动标记目标区域

详细说明​

区域转数值​

将区域对象转换为上下左右四个数值。

使用场景：提取区域边界进行计算或判断。

针脚说明：

  • 区域：要转换的区域对象。
  • 左：输出区域的左边界值。
  • 上：输出区域的上边界值。
  • 右：输出区域的右边界值。
  • 下：输出区域的下边界值。

数值转区域​

使用上下左右四个数值创建区域对象。

使用场景：根据数值动态创建区域。

针脚说明：

  • 左：区域的左边界值。
  • 上：区域的上边界值。
  • 右：区域的右边界值。
  • 下：区域的下边界值。
  • 区域：输出创建的区域对象。

区域偏移​

将区域在水平和垂直方向上移动指定距离。

使用场景：调整区域位置、相对偏移等。

针脚说明：

  • 区域：要偏移的原区域。
  • 偏移：要偏移的距离。
  • 结果：输出偏移后的区域。

位置是否在 区域里​

判断指定位置是否在区域内部。

使用场景：点击位置判断、范围检查等。

针脚说明：

  • 区域：指定的区域范围。
  • 位置：要判断的位置坐标。
  • 结果：输出判断结果。

区域关系​

判断两个区域之间的关系：包含、相交或不相交。

使用场景：区域重叠判断、布局分析等。

针脚说明：

  • 区域(1)：第一个区域。
  • 区域(2)：第二个区域。
  • 关系：输出两个区域的关系类型。
    • 包含：区域1包含区域2。
    • 相交：两个区域相交。
    • 不相交：两个区域不相交。

取区域交集​

计算两个区域相交的部分。

使用场景：提取重叠区域、交集计算等。

针脚说明：

  • 区域(1)：第一个区域。
  • 区域(2)：第二个区域。
  • 区域(右)：输出相交的区域。

取区域中心​

获取区域的中心点坐标。

使用场景：获取区域中心位置进行点击等操作。

针脚说明：

  • 区域：要获取中心的区域。
  • 位置：输出区域的中心点坐标。

取区域随机位置​

在区域内随机选取一个位置。

使用场景：随机点击、模拟人为操作等。

针脚说明：

  • 区域：要选取随机位置的区域。
  • 位置：输出区域内的随机位置坐标。

手动选择区域​

通过用户交互方式在屏幕上选择一个区域。

使用场景：用户手动指定操作区域。

针脚说明：

  • 区域：输出用户选中的区域。

标记区域​

在屏幕上标记一个区域。

使用场景：区域标注、可视化提示等。

针脚说明：

  • 区域：要标记的区域。
