系统

系统类动作用于执行各种系统级操作，如Shell命令、按键模拟、音量控制等。

详细说明​

执行Shell命令​

执行Linux Shell命令，需要root权限。

使用场景：系统级操作、文件操作等。

针脚说明：

  • 命令：要执行的命令字符串。
  • 输出：命令执行的输出结果。
  • 执行(底部)：命令执行成功后的分支。
  • 未达成：命令执行失败后的分支。

模拟按键​

模拟按下系统按键，如返回、主页、音量等。

使用场景：返回上一页、回到主页等。

针脚说明：

  • 按键：要模拟的按键或功能。
    • 返回：返回键。
    • 主页：主页键。
    • 最近列表：打开最近列表。
    • 打开通知：打开通知中心。
    • 打开快捷设置：打开快捷设置。
    • 打开电源菜单：打开电源菜单。
    • 锁屏：锁定屏幕。
    • 截图：调用系统截图。

播放铃声​

播放指定类型的铃声。

使用场景：任务完成提醒、错误提示等。

针脚说明：

  • 铃声：要播放的铃声。
  • 通道：指定播放铃声的通道。

停止铃声​

停止当前正在播放的铃声。

使用场景：取消提醒。

针脚说明：

  • 铃声：要停止的铃声。

文本转语音​

将文本内容转换为语音并播放。

使用场景：语音播报、无障碍功能等。

针脚说明：

  • 文本：要转换为语音的文本。

震动​

让手机震动。

使用场景：触觉反馈、提醒等。

针脚说明：

  • 时长：震动持续时间（毫秒）。

获取音量​

获取当前系统的音量大小。

使用场景：条件判断、音量调节等。

针脚说明：

  • 获取音量：要获取的音量流类型（媒体、铃声音量等）。
  • 当前音量值：当前音量大小。
  • 最大音量值：最大音量大小。

设置音量​

设置系统音量大小。

使用场景：静音操作、音量调节等。

针脚说明：

  • 设置音量：要设置的音量流类型（媒体、铃声音量等）。
  • 音量值：要设置的音量大小。

发送通知​

发送自定义通知到系统通知栏。

使用场景：任务提醒、信息推送等。

针脚说明：

  • 标题：通知的标题文本。
  • 内容：通知的详细内容。
  • 图标：自定义通知图标。
  • 自动取消：点按后是否自动取消通知。
  • 点按执行：通知被点击后执行的连接的逻辑，这段逻辑是完全独立的，相当于执行一个新任务。

发送广播​

发送自定义系统广播。

使用场景：与其他应用通信。

针脚说明：

  • 应用或界面：指定要发送广播的应用与界面。
  • 广播动作：广播的Action字符串。
  • 分类(category)：广播的分类字符串。
  • Uri数据：广播的Uri数据。
  • 参数：广播的数据。

弹出提示​

弹出Toast提示文本。

使用场景：临时提示信息。

针脚说明：

  • 内容：要显示的提示文本。
  • 时长：Toast显示的时间（毫秒）。

复制到剪贴板​

将指定内容复制到系统剪贴板。

使用场景：数据传递、文本复制等。

针脚说明：

  • 文本：要复制到剪贴板的内容，支持文本与图片。

从剪贴板读取​

读取系统剪贴板中的内容。

使用场景：获取复制的文本。

针脚说明：

  • 文本：输出剪贴板中的文本内容。
  • 图片：输出剪贴板中的图片内容。

分享到​

将内容分享到指定的应用。

使用场景：内容分享。

针脚说明：

  • 应用或界面：要分享到的应用或界面。
  • 对象：要分享的内容，支持文本与图片。
  • 作为：指定分享方式。
    • 文字：分享文本内容。
    • 文件：分享文件。

屏幕录制开关​

打开或关闭屏幕录制功能。

使用场景：增强对屏幕变化敏感的任务。

针脚说明：

  • 关/开：开关屏幕录制服务。
  • 等待授权：等待屏幕录制服务授权。

屏幕录制状态​

获取当前屏幕录制是否开启。

使用场景：条件判断。

针脚说明：

  • 结果：是否开启屏幕录制服务。

开关屏幕​

打开或关闭设备屏幕。

使用场景：省电、自动锁屏等。

针脚说明：

  • 开关：点亮或关闭屏幕。

获取屏幕状态​

获取屏幕当前是亮屏、灭屏还是锁定状态。

使用场景：条件判断。

针脚说明：

  • 锁屏状态：输出屏幕当前状态。
    • 息屏：息屏状态。
    • 锁屏：亮屏但未解锁。
    • 解锁：已解锁状态。
  • 分辨率：输出当前屏幕分辨率。
  • 是否竖屏：输出当前屏幕是否为竖屏。

获取电池状态​

获取当前电池电量和充放电状态。

使用场景：低电量处理、充电判断等。

针脚说明：

  • 状态：输出当前充电状态。
    • 未知：未知状态。
    • 充电中：正在充电。
    • 放电中：正在充电，但是充电小于放电的速度。
    • 未充电：未充电。
    • 充满了：充电完成。
  • 电量：输出当前电量百分比。

获取网络状态​

获取当前网络连接状态。

使用场景：网络条件判断。

针脚说明：

  • 状态：网络类型，可选值如下。
    • 无网络：无网络连接。
    • 移动数据：移动数据连接。
    • WIFI：WIFI连接。
    • VPN：VPN连接。

获取WiFi信息​

获取当前连接的WiFi信息。

使用场景：WiFi环境判断。

针脚说明：

  • WiFi名称：输出当前连接的WiFi名称。
  • 网关：输出当前连接的WiFi网关。
  • IP：输出设备当前IP地址。

获取蓝牙 设备​

获取当前已连接的蓝牙设备列表。

使用场景：蓝牙设备判断。

针脚说明：

  • 设备列表：输出已连接的蓝牙设备名称列表。

获取日期​

获取当前日期。

使用场景：日期判断、日期记录等。

针脚说明：

  • 年：当前年份。
  • 月：当前月份。
  • 日：当前日期。
  • 星期：当前星期。

获取时间​

获取当前时间。

使用场景：时间判断、时间记录等。

针脚说明：

  • 时：当前小时（24小时制）。
  • 分：当前分钟。
  • 秒：当前秒钟。
  • 毫秒：当前毫秒。
  • 时间戳：当前时间戳。
