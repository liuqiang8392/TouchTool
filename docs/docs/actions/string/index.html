文本

文本处理类动作用于对文本进行各种操作，包括拼接、截取、匹配、替换等。

动作列表​

动作类型功能说明
转换为文本将任意对象转换为文本
文本转数值提取文本中的第一个数值
文本拼接将多个文本拼接成一个文本
文本截取将文本截取为指定范围的文本
文本正则匹配执行正则匹配，判断是否包含特定文本
文本分割将文本按特定字符分割为多个文本
文本比较判断两个文本是否相等
文本替换将文本中的特定文本替换为另一个文本
文本编码将文本编码为指定编码
文本解码将文本解码为指定编码
文本加密将文本加密为指定编码
OCR识别文本识别图片中指定区域文字
OCR查找文本在图片中查找指定文字
OCR文本是否存在判断图片中是否存在指定文字
JSON解析解析JSON字符串
新建单选创建一个自定义单选

详细说明​

转换为文本​

将任意类型的对象转换为文本格式。

使用场景：数据格式转换、日志输出等。

针脚说明：

  • 对象：要转换的对象。
  • 文本：输出的文本结果。

文本转数值​

从文本中提取数值。

使用场景：从字符串中提取数字进行计算。

针脚说明：

  • 文本：要提取数值的文本。
  • 数值：首个可用数值。
  • 数值列表：所有可用的数值。

文本拼接​

将多个文本片段拼接成一个完整的文本。

使用场景：组合多个变量为完整字符串。

针脚说明：

  • 文本(1)：要拼接的多个文本之一。
  • 文本(2)：要拼接的多个文本之一。
  • 添加针脚：添加新的文本片段。
  • 文本(右)：拼接后的完整文本。

文本截取​

从文本中截取指定范围的子串。

使用场景：提取文本中的部分内容。

针脚说明：

  • 文本：要截取的源文本。
  • 起始位置：截取的起始索引，支持负数索引。
  • 结束位置：截取的结束索引，支持负数索引。
  • 结果：截取后的子串。

文本正则匹配​

使用正则表达式匹配文本，判断是否包含特定模式。

使用场景：复杂文本匹配、数据提取等。

针脚说明：

  • 文本：要匹配的源文本。
  • 匹配文本：用于匹配的正则表达式。
  • 匹配结果：输出所有匹配到的内容段。
  • 捕获结果：输出所有捕获到的内容。

文本分割​

按指定分隔符将文本分割成多个部分。

使用场景：解析CSV数据、拆分字符串等。

针脚说明：

  • 文本：要分割的源文本。
  • 分隔符：用于分割的字符或字符串。
  • 剔除空字符：是否剔除空字符。
  • 正则模式：是否使用正则模式匹配。
  • 文本列表：输出分割后的文本列表。

文本比较​

判断两个文本是否相等。

使用场景：条件判断、文本验证等。

针脚说明：

  • 文本(1)：要比较的文本。
  • 文本(2)：要比较的文本。
  • 忽略大小写：是否区分大小写。
  • 正则模式：是否使用正则模式匹配。
  • 结果：比较结果。

文本替换​

将文本中的指定内容替换为新内容。

使用场景：文本修正、模板替换等。

针脚说明：

  • 文本：要操作的源文本。
  • 查找文本：要替换的文本。
  • 替换文本：替换后的新文本。
  • 正则模式：是否使用正则模式匹配。
  • 替换结果：输出替换后的文本。

文本编码​

将文本编码为指定格式（如Base64、URL编码等）。

使用场景：数据传输、安全存储等。

针脚说明：

  • 编码方式：选择编码方式。
    • URL：将文本编码为URL编码格式。
    • Base64：将文本编码为Base64格式。
  • 文本：要编码的文本。
  • 结果：输出编码后的文本。

文本解码​

将编码的文本解码为原始文本。

使用场景：还原编码数据。

针脚说明：

  • 解码方式：选择解码方式。
    • URL：将文本解码为URL编码格式。
    • Base64：将文本解码为Base64格式。
  • 文本：要解码的文本。
  • 结果：输出解码后的原始文本。

文本加密​

使用指定算法加密文本。

使用场景：敏感数据加密、安全传输等。

针脚说明：

  • 加密方式：选择加密算法。
    • MD5：使用MD5算法加密。
    • SHA-1：使用SHA1算法加密。
    • SHA-256：使用SHA256算法加密。
  • 文本：要加密的文本。
  • 结果：输出加密后的文本。

OCR识别文本​

使用OCR技术识别图片中指定区域的文字。

使用场景：屏幕文字识别、图片内容提取等。

针脚说明：

  • 实时模式：是否启用实时模式，实时模式下每次获取输出值都会进行一次OCR。
  • 图片：要识别的图片。
  • 置信度(左)：识别结果的置信度，范围0-100。
  • 模型：选择使用的OCR模型。
  • 所有文字：输出识别出的文字内容。
  • 文本列表：输出识别出的文字块列表。
  • 区域列表：输出识别出的文字块区域列表。
  • 置信度(右)：输出识别出的文字块的置信度。

OCR查找文本​

在图片中查找指定的文字并返回区域。

使用场景：定位特定文字位置、精准点击等。

针脚说明：

  • 超时：等 待超时时间（毫秒）。
  • 检查间隔：检查OCR间隔时间（毫秒）。
  • 图片：要搜索的图片。
  • 文本(左)：要查找的文字内容。
  • 置信度：识别结果的置信度，范围0-100。
  • 模型：选择使用的OCR模型。
  • 区域：输出匹配到的区域。
  • 文本(右)：输出匹配到的文字内容。
  • 匹配置信度：输出匹配到的文字的置信度。

OCR文本是否存在​

判断图片中是否存在指定的文字。

使用场景：界面状态判断、条件分支等。

针脚说明：

  • 图片：要搜索的图片。
  • 文本：要查找的文字内容。
  • 置信度：识别结果的置信度，范围0-100。
  • 模型：选择使用的OCR模型。
  • 结果：输出是否找到匹配的文字。

JSON解析​

解析JSON字符串，提取其中的数据。

使用场景：API响应解析、配置文件读取等。

针脚说明：

  • 文本：要解析的JSON字符串。
  • 结果：输出解析后的数据。

新建单选​

创建一个自定义的单选选项。

使用场景：用户交互选择、参数配置等。

针脚说明：

  • 选项：可供选择的选项文本列表。
  • 选中项：默认选中的选项。
  • 单选：输出单选对象。
