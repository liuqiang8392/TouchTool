列表

列表操作类动用于创建和操作列表数据结构。

动作列表​

动作类型功能说明
新建列表新建一个列表
列表长度获取列表长度
列表是否为空判断列表是否为空
是否包含元素判断列表是否包含元素
添加元素向列表中添加元素
移除索引元素按索引从列表中移除元素
移除元素对象按对象从列表中移除元素
列表拼接将两个列表拼接
清空列表清空列表
获取元素从列表中获取元素
手动选择元素弹出选择窗口供使用者选择列表的一项
设置元素向列表中设置元素
元素索引获取元素在列表中的索引
获取子列表获取列表中指定范围的子列表
遍历列表遍历列表中的每个元素

详细说明​

新建列表​

创建一个新的列表。

使用场景：初始化列表数据结构。

针脚说明：

  • 对象：创建列表的对象。
  • 添加针脚：向列表中添加元素。
  • 对象列表：输出创建的对象列表。

列表长度​

获取列表中元素的数量。

使用场景：判断列表大小、循环控制等。

针脚说明：

  • 对象列表：要获取长度的列表。
  • 整数：输出列表的元素数量。

列表是否为空​

判断列表是否不包含任何元素。

使用场景：空值检查、条件判断等。

针脚说明：

  • 对象列表：要判断的列表。
  • 结果：输出列表是否为空的结果。

是否包含元素​

判断列表中是 否包含指定元素。

使用场景：元素查找、条件判断等。

针脚说明：

  • 对象列表：要搜索的列表。
  • 对象：要查找的元素。
  • 结果：输出列表是否包含指定元素的结果。

添加元素​

向列表末尾添加一个或多个元素。

使用场景：动态构建列表、数据收集等。

针脚说明：

  • 对象列表：要搜索的列表。
  • 对象：要查找的元素。
  • 元素位置：添加元素的位置，支持负数索引，-1为末尾，0或1为开头。
  • 结果：输出列表是否包含指定元素的结果。

移除索引元素​

根据索引位置移除列表中的元素。

使用场景：按位置删除元素。

针脚说明：

  • 对象列表：要搜索的列表。
  • 索引：移除元素的位置，支持负数索引，-1为末尾，0或1为开头。
  • 结果：输出列表是否正确移除指定元素结果。
  • 对象：输出列表中移除的元素。

移除元素对象​

根据元素值移除列表中的元素。

使用场景：按值删除元素。

针脚说明：

  • 对象列表：要搜索的列表。
  • 对象：要查找的元素。
  • 结果：输出列表是否正确移除指定元素结果。

列表拼接​

将两个列表合并成一个列表。

使用场景：合并多个列表数据。

针脚说明：

  • 对象列表(1)：第一个列表。
  • 对象列表(2)：第二个列表。
  • 结果：输出合并后的列表。

清空列表​

移除列表中的所有元素。

使用场景：重置列表、释放数据等。

针脚说明：

  • 对象列表：要清空的列表。

获取元素​

根据索引获取列表中的元素。

使用场景：读取列表中特定位置的数据。

针脚说明：

  • 对象列表：要搜索的列表。
  • 索引：获取元素的位置，支持负数索引，-1为末尾，0或1为开头。
  • 对象：输出获取的元素值。
  • 结果：输出元素是否存在的结果。

手动选择元素​

弹出选择窗口，让用户从列表中选择一个元素。

使用场景：用户交互选择数据。

记忆位置手动选择

针脚说明：

  • 对象列表：可供选择的列表。
  • 超时：设置选择窗口的超时时间（毫秒），0为无超时。
  • 位置：选择窗口显示位置。
    • 记忆位置：显示在记忆位置的窗口，用户选择后记住位置。
    • 手动选择：设置自定义位置。
      • 窗相对屏位置：指定悬浮窗锚点的相对屏幕锚点的位置。
      • 悬浮窗锚点：指定悬浮窗锚点。
      • 屏幕锚点：指定屏幕锚点。
  • 对象：输出用户选中的元素。

设置元素​

修改列表中指定索引位置的元素值。

使用场景：更新列表中特定位置的数据。

针脚说明：

  • 对象列表：要修改的列表。
  • 索引：要修改元素的索引位置。
  • 对象：要设置的新元素值。
  • 之前的对象：输出修改前的元素值。

元素索引​

获取指定元素在列表中的索引位置。

使用场景：查找元素位置、判断元素是否存在等。

针脚说明：

  • 对象列表：要搜索的列表。
  • 对象：要查找的元素值。
  • 索引：输出元素的索引位置（0表示不存在）。

获取子列表​

从列表中截取指定范围的子列表。

使用场景：提取列表中的部分数据。

针脚说明：

  • 对象列表：要截取的列表。
  • 起始索引：子列表的起始位置（包含）。
  • 结束索引：子列表的结束位置（包含）。
  • 对象：输出截取的子列表。

遍历列表​

对列表中的每个元素执行指定操作。

使用场景：批量处理列表数据。

针脚说明：

  • 跳出循环：执行后在下一轮结束循环。
  • 对象列表：要遍历的列表。
  • 对象：当前遍历的元素值。
  • 索引：当前遍历的元素索引。
  • 执行(底部)：连接要循环执行的逻辑。
  • 完成：循环完成后执行的分支。
